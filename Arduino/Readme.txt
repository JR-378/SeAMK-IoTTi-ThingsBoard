Thingsboard-MQTT contains the code for Arduino Mega, that uploads data to Thingsboard using MQTT connection. Open it with Arduino IDE.

ThingsBoard-Socket contains the code for Arduino Mega, that uploads data to a socket. Open it with Arduino IDE.

libraries.zip contains the required libraries for the code. Unzip it with 7-Zip or Winrar and Copy and paste the folders inside into the Arduino libraries folder, usually located in C:\Users\YOUR-USERNAME\Documents\Arduino\libraries

For information on installing libraries, see: http://www.arduino.cc/en/Guide/Libraries
